## ğŸš€ Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/bensh240/flightHelpers.git
   cd flightHelpers
   ```

2. **Install dependencies**
   ```bash
   npm install
   cd flight-search
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`
